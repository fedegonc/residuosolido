Backup y Restauración de Datos

Gestiona copias de seguridad de toda la información del sistema. Puedes descargar un backup completo en formato JSON y restaurarlo cuando lo necesites. Importante: La restauración eliminará todos los datos actuales y los reemplazará con los del archivo de backup.

Descargar Backup

Genera y descarga un archivo JSON con todos los datos del sistema:

  • Usuarios y sus configuraciones
  • Materiales reciclables
  • Categorías y publicaciones
  • Solicitudes de recolección
  • Feedback de usuarios
  • Configuraciones del sistema

Restaurar Backup

⚠️ Advertencia: Esta acción es irreversible

Al restaurar un backup, todos los datos actuales serán eliminados permanentemente y reemplazados con los datos del archivo. Asegúrate de haber descargado un backup reciente antes de continuar.

Requisitos del archivo:

  • Formato: JSON (.json)
  • Tamaño máximo: 50 MB
  • Debe ser un backup generado por este sistema
  • Versión compatible: 1.0

Información Adicional

📋 Recomendaciones:

  • Descarga backups periódicamente (diario, semanal o mensual según tus necesidades)
  • Almacena los backups en un lugar seguro fuera del servidor
  • Verifica que el archivo descargado no esté corrupto antes de eliminarlo
  • Realiza pruebas de restauración en un ambiente de desarrollo antes de producción
  • Documenta la fecha y hora de cada backup para facilitar la recuperación
