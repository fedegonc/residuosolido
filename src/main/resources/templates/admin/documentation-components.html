Componentes UI

Biblioteca de componentes reutilizables con Thymeleaf y Tailwind CSS

Sistema de componentes

Esta biblioteca de componentes proporciona fragmentos Thymeleaf reutilizables para mantener una apariencia consistente en toda la aplicación. Los componentes utilizan exclusivamente Tailwind CSS y están diseñados para ser fácilmente incorporados en cualquier página.

Botones

Incluye los botones usando th:replace o th:insert en tus templates.

Botón primario

<div th:replace="~{fragments/components/buttons :: primary('Texto del botón', '/ruta/destino')}"></div>

Usar para acciones principales y positivas como crear, guardar o confirmar.

Botón secundario

<div th:replace="~{fragments/components/buttons :: secondary('Texto del botón', '/ruta/destino')}"></div>

Usar para acciones secundarias como cancelar o volver.

Botón de peligro

<div th:replace="~{fragments/components/buttons :: danger('Texto del botón', '/ruta/destino')}"></div>

Usar para acciones destructivas como eliminar o cancelar operaciones irreversibles.

Botón con icono

<div th:replace="~{fragments/components/buttons :: iconLeft('Texto del botón', '/ruta/destino', 'plus')}"></div>

Iconos disponibles: plus, edit, trash

Botones de formulario

<div th:replace="~{fragments/components/buttons :: primaryButton('Guardar')}"></div>
<div th:replace="~{fragments/components/buttons :: secondaryButton('Cancelar')}"></div>
<div th:replace="~{fragments/components/buttons :: dangerButton('Eliminar')}"></div>

Alertas

Alertas para mostrar mensajes de éxito, error, advertencia e información.

Alerta de éxito

<div th:replace="~{fragments/components/alerts :: success('Operación completada correctamente.')}"></div>

Alerta de error

<div th:replace="~{fragments/components/alerts :: error('No se pudo completar la operación.')}"></div>

Alerta de advertencia

<div th:replace="~{fragments/components/alerts :: warning('Esta acción podría tener consecuencias.')}"></div>

Alerta de información

<div th:replace="~{fragments/components/alerts :: info('Información relevante para el usuario.')}"></div>

Alerta flash automática

Esta alerta desaparece automáticamente después de 5 segundos.

<div th:replace="~{fragments/components/alerts :: flash('Este mensaje desaparecerá en 5 segundos', 'success')}"></div>

Tipos disponibles: success, error, warning, info

Uso en controladores

Mensajes flash en controladores

@PostMapping("/admin/categories")
public String adminSaveCategory(
        @RequestParam(value = "action", required = false) String action,
        @ModelAttribute Category category,
        RedirectAttributes redirectAttributes) {
    try {
        // ... lógica del controlador ...
        
        // Mensaje de éxito
        redirectAttributes.addFlashAttribute("successMessage", "Categoría creada correctamente");
        
        // También se puede usar errorMessage, warningMessage, infoMessage
    } catch (Exception e) {
        redirectAttributes.addFlashAttribute("errorMessage", "Error al procesar categoría: " + e.getMessage());
    }
    
    return "redirect:/admin/categories";
}

Mostrar mensajes en la plantilla

<!-- Mostrar mensajes flash en cualquier plantilla -->
<div th:if="${successMessage}" th:replace="~{fragments/components/alerts :: success(${successMessage})}"></div>
<div th:if="${errorMessage}" th:replace="~{fragments/components/alerts :: error(${errorMessage})}"></div>
<div th:if="${warningMessage}" th:replace="~{fragments/components/alerts :: warning(${warningMessage})}"></div>
<div th:if="${infoMessage}" th:replace="~{fragments/components/alerts :: info(${infoMessage})}"></div>
